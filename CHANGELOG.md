# v2.4 - 2024/12/31

# 変更点
- 引数オプションを追加
  - -i, --input：  ファイル名を指定。指定したファイル名以外はスキップする。ファイル名の一部でOK。
  - -f, --force-save-text：  PDF解析結果をテキストファイルに保存する。通常エラー発生時のみ保存を行うが、このオプションを指定するとテキストファイルが存在しない場合、必ず保存する。
- 国内株式のページ数チェック機能を実装
- pdfを解析したテキストファイル内のU+000C(\f) Form feedを半角スペースに置換するようにした。
